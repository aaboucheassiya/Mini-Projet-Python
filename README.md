# Mini-Projet Python – Analyse de données

# Description du projet
Ce projet est un mini-projet Python réalisé dans le cadre du module Cloud Computing et Outils Collaboratifs.  
L’objectif principal est de mettre en pratique :
- La structuration d’un projet Python
- L’utilisation de Git et GitHub
- Le versionnement avec des commits clairs
- La rédaction d’une documentation professionnelle
# Technologies utilisées
- Python 3.x
- Git
- GitHub  
-Bibliothèques Python :
  - matplotlib
  - numpy
